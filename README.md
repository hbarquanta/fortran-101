# Fortran 101 Workshop

This repository is set up for our Fortran workshop. It contains a small starter project, example code, and a first exercise so you can begin coding right away.

## Project layout

- src/ — sample Fortran programs
- exercises/ — workshop exercises
- bin/ — compiled binaries (generated by the build step)

## Build and run the first example

Run the following commands from the repository root:

```sh
make
./bin/01_hello
```

## Compiling by hand (without make)

```sh
gfortran -o bin/mytest src/types.f90 src/test.f90
./bin/mytest
```

List every source file the program needs, in order (a file with a
`module` must come before a file that uses it). `-o bin/mytest` names
the output binary.

## Suggested workflow

- Keep each exercise in its own folder or file.
- Use clear variable names and add comments when something is non-obvious.
- Build often and test small changes incrementally.
